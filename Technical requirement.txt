Тестовое задание: лента Twitter 
Стек:
•	Node.js 6+
•	Angular 4
•	Angular Material
•	Дополнительно: PostgreSQL или MySQL

Цель
Создание веб-приложение с следующим функционалом: 

Получение и отображение твитов 
Приложение должно выводить твиты (twitter.com) по некоторым ключевым словам (например, “eCommerce” и “video”). Оформление приложения должно соответствовать material design.
Развертывание
Необходимо разместить live-версию приложения в облачном хостинге (Heroku, AWS, OpenShift  etc.) с бесплатным тарифным планом
Дополнительно: Оценка твитов 
Реализовать возможность оценки твитов (like или star rate), после чего они будут сохранены в базу данных приложения. 
Дополнительно: Возможность просмотра оцененных твитов 
Создать отдельное представление для отображения оцененных твитов 
Вы можете реализовать это в отдельном окне, табах или как-либо ещё на Ваше усмотрение.

Примечания: 
•	При разработке необходимо придерживаться git-flow. 
•	Исходный код приложения должен быть размещён в публичный репозиторий(GitHub, GitLab, Bitbucket)
•	В проекте обязательно должен быть README с описанием, инструкцией по установке и ссылкой на live версию.
•	Вы можете использовать любые другие библиотеки или компоненты, которые понадобятся Вам для выполнения задания
•	Для работы с базой данных используйте ORM систему.

Deadline 
Выполнение данного задание рассчитано на 3 календарных дня. В случае, если у Вас возникли проблемы или задержки в выполнении задания, просто сообщите о смещении сроков.
